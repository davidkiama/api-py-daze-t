# 🚀 Daze-T Crypto Payment API

Production-ready **Flask backend** for handling cryptocurrency payments via NOWPayments, storing transactions in MySQL, sending Telegram alerts, and processing contact form emails.

This API powers the **Daze-T** crypto payment platform.

---

# 🔗 Related Project (Frontend)

The React frontend for this API is available here:

👉 **Frontend Repository:**  
https://github.com/davidkiama/Daze-T

The backend and frontend work together to create, monitor, and complete crypto payments.

---

# 📌 Features

- ✅ Create crypto payment invoices
- ✅ Poll & verify payment status
- ✅ Store transactions in MySQL
- ✅ Telegram alerts for payment events
- ✅ Email contact form (SMTP)
- ✅ Secure CORS configuration
- ✅ Success redirect to frontend

---

# 🏗 Tech Stack

- Python 3
- Flask
- SQLAlchemy
- MySQL
- Requests
- python-dotenv
- Telegram Bot API
- SMTP (Email)
